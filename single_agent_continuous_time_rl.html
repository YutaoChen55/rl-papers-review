Reinforcement Learning in Continuous Time and Space: A Stochastic Control Approach

Haoran Wang, Thaleia Zariphopoulou, Xun Yu Zhou | Journal of Machine Learning Research | JMLR 21(198):1-34

研究问题与动机

传统强化学习方法通常需要对时间、状态和动作空间进行先验离散化，这在连续时间连续空间（CTCS）问题中会导致：

  • 维度灾难：离散化网格随维度指数增长
  • 信息损失：连续动态被粗糙近似
  • 探索-利用困境：缺乏系统化的探索机制理论分析

本文首次在连续时间框架下建立RL的严格数学基础，通过探索性随机控制和熵正则化实现探索-利用的最优权衡。

数学模型建立

状态动态（受控扩散过程）

dXt = b(Xt, at)dt + σ(Xt, at)dWt,   X0 = x

其中 at ∈ A 是控制（动作），Wt 是标准布朗运动

探索性控制问题（熵正则化）

不同于传统RL使用确定性策略，本文引入分布值控制（distributional control）π(·|t,x) ∈ P(A)，表示在状态x时刻t选择动作a的概率分布。

J(x; π) = E[∫0T (∫A r(Xt, a)π(a|t,Xt)da - λ·H(π(·|t,Xt)))dt + h(XT)]

其中 H(π) = -∫A π(a)log π(a)da 是策略的熵，λ > 0 是温度参数

主要定理与理论结果

定理 1（探索性HJB方程）

值函数 V(t,x) = supπ J(t,x; π) 满足以下非线性PDE：

∂V/∂t + supπ∈P(A) {∫A [b(x,a)·∇V + ½tr(σσT∇²V) + r(x,a)]π(a)da + λH(π)} = 0

边界条件：V(T,x) = h(x)

定理 2（Gibbs最优策略）

最优分布值策略具有Gibbs（Boltzmann）形式：

π*(a|t,x) = exp{(1/λ)[b(x,a)·∇V + ½tr(σσT∇²V) + r(x,a)]} / Z(t,x)

其中 Z(t,x) = ∫A exp{(1/λ)[...]}da 是归一化常数。温度λ控制探索程度：λ→0时退化为贪婪策略，λ→∞时趋于均匀探索。

定理 3（值函数表示）

值函数可表示为以下线性抛物型PDE的解：

∂V/∂t + λ·log ∫A exp{(1/λ)[b(x,a)·∇V + ½tr(σσT∇²V) + r(x,a)]}da = 0

该方程称为熵正则化HJB方程或软HJB方程。

主要贡献与意义

理论贡献

  • • 建立连续时间RL的严格数学框架
  • • 通过熵正则化实现探索-利用最优权衡
  • • 导出Gibbs形式的最优探索策略
  • • 将问题转化为可解的线性PDE

算法意义

  • • 避免时间/空间离散化
  • • 为后续TD学习、Q学习提供理论基础
  • • 温度参数λ提供可调节的探索机制
  • • 连接随机控制与强化学习

Policy Evaluation and Temporal-Difference Learning in Continuous Time and Space: A Martingale Approach

Yanwei Jia, Xun Yu Zhou | Journal of Machine Learning Research | JMLR 23(154):1-55

研究问题与动机

策略评估（Policy Evaluation, PE）是RL的核心子问题：给定一个固定策略，估计其值函数。在连续时间框架下，传统方法面临：

  • 离散化误差：时间离散化导致TD学习偏差
  • 理论空白：缺乏连续时间TD算法的收敛性保证
  • 算法设计：如何直接从连续时间数据学习？

本文提出鞅方法（Martingale Approach），建立连续时间PE的统一理论框架。

数学模型建立

值函数定义

对于给定策略a(·)，定义状态值函数：

Va(x) = E[∫0T r(Xt, a(Xt))dt + h(XT) | X0 = x]

其中Xt服从策略a下的动态：dXt = b(Xt, a(Xt))dt + σ(Xt, a(Xt))dWt

鞅条件（核心创新）

定义过程 Mt = Va(Xt) + ∫0t r(Xs, a(Xs))ds，则：

Mt 是鞅 ⟺ Va 满足无穷小生成元方程：AaVa + ra = 0

其中Aa = b(x,a)·∇ + ½tr(σσT∇²) 是策略a下的无穷小生成元

主要定理与TD算法

定理 1（鞅特征化）

值函数Va是以下问题的唯一解：

AaV + ra = 0,   x ∈ ℝd
V(x) = h(x),   x ∈ ∂𝒪（边界条件）

该椭圆型PDE称为泊松方程或值函数方程。

定理 2（连续时间TD学习）

定义TD误差（无穷小形式）：

δt = r(Xt, a(Xt))dt + dV(Xt) = [r(Xt, a(Xt)) + AaV(Xt)]dt + dMt

其中dMt = ∇V·σdWt是鞅噪声。参数更新规则：

dθt = -αt ∇θVθ(Xt) · δt

定理 3（收敛性）

在适当条件下（步长αt满足Robbins-Monro条件：∑αt = ∞, ∑αt² < ∞），连续时间TD算法以概率1收敛到真实值函数。

算法实现与贡献

连续时间TD(0)

θk+1 = θk - αk ∇θVθ(Xtk) · [rΔt + Vθ(Xtk+1) - Vθ(Xtk)]

基于连续时间轨迹的在线更新

连续时间LSTD

θ* = A-1b, 其中 A = E[∇V∇VT], b = E[∇V·r]

最小二乘时序差分

Policy Gradient and Actor-Critic Learning in Continuous Time and Space: Theory and Algorithms

Yanwei Jia, Xun Yu Zhou | SSRN Electronic Journal | 2021

研究问题与动机

策略梯度（Policy Gradient, PG）是RL中策略优化的重要方法。在连续时间框架下：

  • 如何将离散时间的PG理论扩展到连续时间？
  • 如何设计连续时间的Actor-Critic架构？
  • 如何结合探索性控制框架？

本文基于前文的鞅方法和探索性控制框架，建立连续时间PG的严格理论。

数学模型建立

参数化策略

考虑参数化策略πθ(a|t,x)，其中θ ∈ ℝp是策略参数。目标是：

maxθ J(θ) = Eπθ[∫0T (∫A r(Xt, a)πθ(a|t,Xt)da - λH(πθ))dt + h(XT)]
主要定理与PG公式

定理 1（策略梯度定理 - 连续时间）

策略梯度可表示为：

∇θJ(θ) = Eπθ[∫0T ∫A (Qπθ(t,Xt,a) - λlog πθ(a|t,Xt) - λ) ∇θlog πθ(a|t,Xt) · πθ(a|t,Xt)da dt]

其中Q函数定义为：Qπ(t,x,a) = Eπ[∫tT(r-λH)ds + h | Xt=x, at=a]

定理 2（Actor-Critic架构）

Critic更新（值函数估计）
dθV = -αt ∇θVVθV(Xt) · δt

使用连续时间TD学习

Actor更新（策略优化）
dθπ = βt ∇θπlog πθπ(at|t,Xt) · AθV(t,Xt,at)

q-Learning in Continuous Time

Yanwei Jia, Xun Yu Zhou | Journal of Machine Learning Research | JMLR 24(161):1-61

研究问题与动机

Q-Learning是最流行的无模型RL算法之一。在连续时间框架下：

  • 如何定义连续时间的Q函数？
  • 如何导出连续时间的Bellman最优性方程？
  • 如何设计连续时间的Q学习算法？

本文建立连续时间Q学习的完整理论框架，并连接SARSA、PG等方法。

数学模型建立

连续时间Q函数定义

不同于离散时间的Q(s,a)，连续时间Q函数Q(t,x,a)依赖于时间t、状态x和瞬时动作a：

Q(t,x,a) = E[∫tT r(Xs, as)ds + h(XT) | Xt=x, at=a, 最优控制]
主要定理与Q学习算法

定理 1（连续时间Bellman最优性方程）

最优Q函数满足：

∂Q/∂t + supa∈A {b(x,a)·∇xQ + ½tr(σσT∇x²Q) + r(x,a)} = 0

边界条件：Q(T,x,a) = h(x)

定理 2（Q学习更新规则）

定义连续时间TD误差：

δt = r(Xt, at)dt + supa' Q(t+dt, Xt+dt, a') - Q(t, Xt, at)

Q函数参数更新：

dθt = -αt ∇θQθ(t, Xt, at) · δt

定理 3（与SARSA和PG的联系）

SARSA解释

当使用当前策略采样动作a'而非max时，Q学习退化为SARSA

PG恢复

在适当参数化下，Q学习更新可恢复PG算法
