Control Barrier Functions (CBF) 核心理论文献综述

Control Barrier Function Based Quadratic Programs for Safety Critical Systems
作者: Aaron D. Ames, Samuel Coogan, Magnus Egerstedt 等 期刊: IEEE Transactions on Automatic Control 年份: 2017 DOI: 10.1109/TAC.2016.2638961
奠基性工作 CBF理论 安全控制
研究背景与动机
传统控制理论主要关注系统的稳定性（Stability），但在许多安全关键应用中（如自动驾驶、机器人操作），保证系统始终处于安全状态集内比单纯稳定更为重要。控制障碍函数（Control Barrier Function, CBF）提供了一种系统化的方法来保证安全性。
核心数学模型
考虑仿射控制系统：
ẋ = f(x) + g(x)u
定义安全集 C = {x ∈ ℝⁿ : h(x) ≥ 0}，其中 h(x) 是连续可微函数。CBF 要求存在扩展类 K 函数 α 使得：
L_f h(x) + L_g h(x)u + α(h(x)) ≥ 0
其中 L_f h 和 L_g h 分别是 h 沿 f 和 g 的李导数。
主要贡献
  • 提出了基于 CBF 的二次规划（QP）框架，将安全性约束编码为控制输入的约束
  • 证明了在满足 CBF 条件下，系统状态将始终保持在安全集 C 内
  • 将 CBF 与控制李雅普诺夫函数（CLF）结合，同时保证安全性和稳定性
  • 提供了连续时间系统的严格数学保证
CBF-CLF 联合优化
核心优化问题形式：
min_u ½‖u - u_ref‖²
s.t. L_f V(x) + L_g V(x)u + γV(x) ≤ 0 (稳定性)
     L_f h(x) + L_g h(x)u + α(h(x)) ≥ 0 (安全性)
其中 V(x) 是控制李雅普诺夫函数，h(x) 是控制障碍函数。
A Framework for Worst-Case and Stochastic Safety Verification Using Barrier Certificates
作者: Stephen Prajna, Ali Jadbabaie, George J. Pappas 期刊: IEEE Transactions on Automatic Control 年份: 2007 DOI: 10.1109/TAC.2007.902736
先驱工作 安全验证 障碍证书
研究背景
在安全性验证中，传统的可达性分析计算复杂度极高。本文提出了"障碍证书"（Barrier Certificates）的概念，通过构造一个 Lyapunov-like 函数来证明系统的安全性，避免了复杂的可达集计算。
障碍证书定义
对于系统 ẋ = f(x)，若存在函数 B(x) 满足：
B(x) ≤ 0 ∀x ∈ X₀ (初始集)
B(x) > 0 ∀x ∈ Xᵤ (不安全集)
∂B/∂x · f(x) ≤ 0 ∀x (沿轨迹不增)
则系统从 X₀ 出发的轨迹永远不会进入 Xᵤ。
主要贡献
  • 首次系统性地提出了障碍证书方法用于安全验证
  • 将问题转化为 SOS（Sum of Squares）规划，可用凸优化求解
  • 扩展到了随机系统，处理不确定性下的安全性
  • 为后续 CBF 的发展奠定了理论基础
与 CBF 的关系：Prajna 的 Barrier Certificates 主要关注验证（Verification），而 Ames 的 CBF 关注综合（Synthesis）。CBF 可以看作是 Barrier Certificates 在控制综合方向的扩展。
Control Barrier Functions: Theory and Applications
作者: Aaron D. Ames, Samuel Coogan, Magnus Egerstedt 等 会议: European Control Conference (ECC) 年份: 2019 DOI: 10.23919/ECC.2019.8796030
综述/tutorial 理论扩展 应用案例
论文定位
本文是 CBF 领域的 Tutorial 论文，系统性地总结了 CBF 的理论框架和各类应用。适合作为入门和综述阅读。
理论扩展
  • 高阶 CBF：处理相对度大于1的系统
  • 时变 CBF：处理时变安全约束
  • 输入约束 CBF：考虑控制输入饱和
  • 鲁棒 CBF：处理模型不确定性
  • 指数 CBF：提供指数收敛的安全保证
应用领域
  • 自动驾驶：车道保持、避障、交叉口通行
  • 多机器人系统：碰撞避免、编队保持
  • 机械臂：关节限位、奇异点避免
  • 无人机：空域保持、障碍物规避
  • 行人导航：人群中的安全导航
高阶 CBF 公式
对于相对度 r > 1 的系统，定义高阶 CBF：
ψ₀(x) = h(x)
ψ₁(x) = ẏ₀(x) + α₁(ψ₀(x))
...
ψᵣ(x) = ẏᵣ₋₁(x) + αᵣ(ψᵣ₋₁(x)) ≥ 0
通过递归构造，将高阶约束转化为可执行的控制约束。
Multi-Agent Safe Planning with Gaussian Processes
作者: Lukas Brunke, Melissa Greeff, Adam W. Hall 等 会议: IEEE/RSJ IROS 年份: 2022 DOI: 10.1109/IROS47612.2022.9981112
多智能体 GP模型 安全规划
研究动机
多智能体系统中的安全规划面临两大挑战：1) 环境不确定性；2) 其他智能体的行为不确定性。本文结合高斯过程（Gaussian Processes）和 CBF，实现了数据驱动的安全规划。
核心方法
使用 GP 学习系统动力学的不确定性：
ẋ = f(x) + g(x)u + d(x)
d(x) ~ GP(0, k(x,x'))
构建概率 CBF 约束，以高置信度保证安全性。
主要贡献
  • 将 GP 与 CBF 结合，处理学习模型的不确定性
  • 提出了概率安全保证框架
  • 在多无人机系统中验证了方法的有效性
  • 实现了实时安全规划
Multi-Objective Compositions for Collision-Free Connectivity Maintenance in Teams of Mobile Robots
作者: Lars Lindemann, Dimos V. Dimarogonas 会议: IEEE CDC 年份: 2018 DOI: 10.1109/CDC.2018.8619705
多目标 CBF组合 连通性保持
研究问题
多机器人系统需要同时满足多个安全约束：1) 机器人间避碰；2) 通信连通性保持；3) 障碍物避免。如何将这些约束有效地组合起来？
CBF 组合方法
对于多个 CBF h₁, h₂, ..., hₘ，定义组合 CBF：
H(x) = min{h₁(x), h₂(x), ..., hₘ(x)} (最小值组合)
或
H(x) = Σᵢ wᵢ hᵢ(x) (加权组合)
不同的组合方式对应不同的安全语义。
主要贡献
  • 提出了系统化的 CBF 组合框架
  • 分析了不同组合方式的保守性
  • 实现了碰撞避免与连通性保持的联合保证
  • 在移动机器人团队中验证了方法

📊 CBF 研究脉络总结

理论基础

Prajna 2007 提出 Barrier Certificates 概念，Ames 2016-2017 将其扩展到控制综合领域，形成 CBF 理论框架。

核心方法

通过构造障碍函数 h(x)，将安全约束编码为控制输入的线性约束，通过 QP 实时求解。

扩展方向

高阶 CBF、鲁棒 CBF、随机 CBF、多智能体 CBF、与 RL 结合等。

应用场景

自动驾驶、机器人操作、无人机群、多智能体系统、电力系统等安全关键领域。

🔑 关键公式速查

CBF 条件：L_f h + L_g h·u + α(h) ≥ 0

CLF-CBF QP：min ‖u‖² s.t. CLF & CBF 约束

安全集：C = {x | h(x) ≥ 0}
